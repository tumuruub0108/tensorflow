# Scalar
A scalar is just a single number.
No dimensions (rank 0 tensor in TensorFlow).
Shape: []

ğŸ‘‰ Examples:
    5, 3.14, e, temperature = 37.

    import tensorflow as tf

    scalar = tf.constant(5)
    print(scalar)        # tf.Tensor(5, shape=(), dtype=int32)
    print(scalar.shape)  # ()


# Vector
A vector is an ordered list of numbers (1D array).
    One dimension (rank 1 tensor in TensorFlow).
    Shape: [n]

It is similar to one-dimensional arrays. 
    
ğŸ‘‰ Examples:
    A point in 3D space -> [x, y, z] = [2, 5, -1]
    Velocities, embeddings, features

vector = tf.constant([1, 2, 3])
print(vector)        # tf.Tensor([1 2 3], shape=(3,), dtype=int32)
print(vector.shape)  # (3,)

ğŸ”¹ Key Difference

Scalar â†’ just a number (no direction, no magnitude beyond itself).
Vector â†’ collection of numbers that can represent direction and magnitude in space.

# Matrix
A matrix is a 2D array of numbers arranged in rows and columns.
    Rank = 2 (two axes: rows and columns)
    Shape = [m, n]

ğŸ‘‰ Example:
    [
    1 2 3 
    4 5 6
    ]

This has shape (2, 3) â†’ 2 rows, 3 columns.
import tensorflow as tf

matrix = tf.constant([[1, 2, 3], [4, 5, 6]])
print(matrix)        # shape=(2, 3)
print(matrix.shape)  # (2, 3)



# Shape
In TensorFlow, a shape refers to the dimensions of a tensor â€” essentially the â€œstructureâ€ or â€œsizeâ€ of the data stored inside that tensor.

# What is a Tensor?
A tensor is just a multi-dimensional array (like a generalization of scalars, vectors, and matrices).
    Scalar â†’ rank 0 tensor â†’ shape []
    Vector â†’ rank 1 tensor â†’ shape [n]
    Matrix â†’ rank 2 tensor â†’ shape [m, n]
    Higher dimensions â†’ rank k tensor â†’ shape [d1, d2, ..., dk]

# Shape in TensorFlow
    The shape describes:
        Rank (number of dimensions)
        Size of each dimension
Example

# Scalar
s = tf.constant(5)
print(s.shape)  # ()

# Vector
v = tf.constant([1, 2, 3])
print(v.shape)  # (3,)

# Matrix
m = tf.constant([[1, 2, 3], [4, 5, 6]])
print(m.shape)  # (2, 3)
